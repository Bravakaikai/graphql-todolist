# graphql_todolist
Vue.js with GraphQL

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
Get data by subscription.
```
npm run serve
```

### Compiles and hot-reloads for development
Get data by query.
```
npm run serve_query
```

### Compiles and minifies for production
```
npm run build
```
